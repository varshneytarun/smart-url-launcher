Quick Guide

  • Pattern: Regular expression. Example: INC\\d{7,}
  • URL: Template with {value}. Example: https://ex.com/{value}
  • ServiceNow: INC\\d{7,} → https://instance.service-now.com/nav_to.do?uri=incident.do?sysparm_query=number={value}
  • Jira: [A-Z]{2,}-\\d+ → https://domain.atlassian.net/browse/{value}
  • GitHub: #\\d+ → https://github.com/owner/repo/issues/{value}

Keyboard Shortcut

You can use the configured keyboard shortcut to trigger the extension from anywhere. Useful when you have copied patterns from a document or webpage.

You can customize this from the Settings tab → Keyboard Shortcut.

Reset to Default Patterns?
